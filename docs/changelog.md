# Versioning

Pulse follows Semantic Versioning.

```
MAJOR.MINOR.PATCH
```

Examples

```
1.0.0

1.1.0

1.1.1
```

Meaning

Major

Breaking changes

Minor

New features

Patch

Bug fixes
